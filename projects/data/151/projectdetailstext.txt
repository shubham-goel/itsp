<h3>Web Server</h3>
<ul>
	<li> Web Server provides an interface to the user to control the bot remotely from any place in the world.</li>
	<li>It has basic commands of movement for bot in front, back, right and back and also the its movement at any desired angle.</li>
	<li>it has basic commands of rotation in anticlockwise or clockwise.</li>
</ul>
<h3>Android programming</h3>
<ul>
	<li>Making the app is probably the toughest part of the project</li>
	<li>App has two mode.
	<ul>
		<li>Manuel Mode :-    user can control the bot directly using the mobile phone via bluetooth.</li>
		<li>Automatic mode :- user control the  bot using web server. also user click picture . </li>
	</ul>
	</li>
</ul>
